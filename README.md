# A-Roblox-Zombie-Game
Domain expansion, Fire-breathing jutsu, the inverted spear of heaven, with various weapons from different anime to fight zombified versions of your favorite anime charaters!

# Day 1 - Create the first zombie NPC
Deciding on the start of the project, I chose to start with the enemies element(NPC) of the game starting with the first mob. To make my first mob, it required an insight on what world would the first battlefield take place in and since this game is anime themed, I chose Dragon Ball as the first world, as it is known to be the grandfather of shounen anime, a popular genre many are interested in. 

Making the Goku zombie was not hard, but due to being new, understanding the code software took a bit of time. Starting with making the new character, I decided to use a R6 model, which is basic in animation and doesn't have a greater rotation distance. Decorating the NPC took a tutorial since I was new to the software. As I finished that tutorial, I moved on to getting the NPC to move. But I didn't know how to even code in script as I am unfamiliar with the language. However, after watching this youtube tutorial https://www.youtube.com/watch?v=mKIUsz81JLE, I was able to create a tracking NPC with an attacking animation.
